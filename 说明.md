# 测试数据标签生成与 Qwen3.6-27B 微调训练说明文档

## 一、 整体流程概述
本流程主要包含**数据标签伪标记**、**数据混洗与增强（Data Augmentation）**以及**大模型多卡 LoRA 微调（SFT）**三个核心阶段。目的在于通过结合特定领域 API 标注数据与通用采样数据，提升 Qwen3.6-27B 模型在多模态化学结构理解与相关任务上的表现。

---

## 二、 核心执行步骤核查

### 阶段 1：测试数据伪标签获取（Pseudo-labeling）
* **执行脚本**：`get_pseudo_label.py`
* **调用接口**：`https://ocsr.dp.tech/mol/img2mol`（DeepPotential 图像转分子结构接口）
* **核查要点**：
    * 检查输入测试数据的完整性与图像质量。
    * **确认接口调用成功计数**：必须确保有 **3,700 条** 数据成功获取到对应的分子式/结构式标签。

### 阶段 2：数据集构建与混洗
* **数据组成**：
    1. **特定领域数据**：上一步成功获取标签的 **3,700 条** 测试数据。
    2. **通用增强数据**：`MolParser-7M` 抽样数据集。
* **融合结果**：两部分数据合并后，最终生成总体量约为 **11 万条（~11w）** 的多模态训练数据集。
* **核查要点**：
    * 检查 `MolParser-7M` 的采样分布与覆盖面，避免由于单一数据源导致模型产生偏置。
    * 确保两部分数据在合并后进行了**全量随机混洗（Shuffle）**，防止模型在训练早期对某一类数据产生过拟合。
    * 检查数据 json 格式，确保图像/视频路径与对应的文本标签严格对齐。

### 阶段 3：多卡并行的 LoRA 微调（SFT）
* **微调基座**：`Qwen3.6-27B` 
* **微调方式**：LoRA（Low-Rank Adaptation）高效参数微调。
